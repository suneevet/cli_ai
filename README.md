# CLI AI 🚀

Удобный и быстрый AI-ассистент, работающий прямо в вашем терминале. Программа использует API OpenRouter для общения с современными языковыми моделями, поддерживает сохранение контекста беседы и имеет встроенные команды управления.

## ✨ Особенности
* 💬 Сохранение контекста: программа помнит историю диалога (до 5 последних сообщений для экономии токенов).
* 🔄 Стриминг ответов: текст выводится на экран по буквам в реальном времени с красивым форматированием (благодаря библиотеке rich).
* 📅 Быстрые команды: встроенное распознавание вопросов о текущей дате и времени без траты лимитов API.
* 🛠 Консольные команды:
  * .clear — очистить экран терминала.
  * .reset — сбросить историю диалога и начать заново.
  * .help — показать справку по командам.
  * .exit — закрыть программу.

## 📦 Как запустить готовую версию (.exe)
Если вы не хотите устанавливать Python, вы можете скачать уже скомпилированный файл:
1. Перейдите в раздел Releases на этой странице GitHub и скачайте файл cli_ai.exe.
2. Создайте в одной папке с файлом cli_ai.exe текстовый файл с именем .env.
3. Откройте его через Блокнот и добавьте вашу строчку с API-ключом:
`text
   OPENROUTER_API_KEY=ваш_ключ_api
